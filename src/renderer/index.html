Automation is best-effort for this 7-click UI path; if it didn't finish, finish manually and click "Elements Ready — Continue."

    3. 
              
  3. Click the "Image" tab on the left
    4. 
              
  4. Ensure the model dropdown shows "Cinema 2.0"
    5. 
              
  5. Click the @ icon (rightmost in prompt toolbar)

Cinematic Mode — Scene Images Need Manual Generation

          

            Cinema Studio 2.0 automation couldn't complete some scene images (selector / picker fragility). Generate each one manually and save it to the listed path. The pipeline will rescan disk and continue when you click the button below.
          


          


          

            Manual generation steps (click to expand)
            
    
              
  1. Open Higgsfield → Cinema Studio → click "2.5" toggle (top-right)
    2. 
              
  2. Click "Image" tab on the left → ensure model dropdown shows "Cinema 2.0"
    3. 
              
  3. For each scene listed above:
                
      
                  
    • Click + (left of prompt) → "Image Generations" tab → pick the location reference image listed
      • 
                  
    • Type a blocking prompt: "WIDE SHOT inside [location] from the reference image. @character_name [position]. @character_name [position]. [lighting notes]. Photorealistic cinematic still."
      • 
                  
    • Use @ autocomplete for character refs (type @ + 2-3 letters + Enter)
      • 
                  
    • Set 16:9 Cinematic, 4K → Generate
      • 
                  
    • Right-click the result → Download → save to the "Save to" path listed for that scene
      • 
                
    
              
    4. 
              
  4. When all scene images exist on disk at their listed paths, click the button below
    5.
